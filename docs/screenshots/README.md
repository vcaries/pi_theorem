# Screenshots & demo media

Add your captures here and reference them from the main `README.md`:

- `light.png` — the app in light theme.
- `dark.png` — the app in dark theme.
- `demo.gif` — a short walkthrough (load the Chen example → Compute → results).

`preview.svg` is a lightweight mockup used as a placeholder until real captures
are added.

## Tips for good demo media

- Record at a 16:10 or 16:9 aspect ratio, ~1280px wide.
- Keep GIFs under ~8 MB (use a tool like `gifski` or `ffmpeg`).
- Show the flagship flow: load **Chen (1990)** → **Compute Π groups** → scroll
  the eight resulting groups; then toggle dark mode and switch FR/EN.
